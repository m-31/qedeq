Release Contents

To run this application at least Java 1.4 is required. To produce PDF files you need LaTeX. You should install a correction for the buggy LaTeX package longtable. One possibility is http://www.cs.rutgers.edu/~ccshan/longtable/ a patch from Chung-chieh Shan.
