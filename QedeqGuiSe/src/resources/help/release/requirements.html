Requirements

Java 5 or later is required, Java 6 is recommended.

The underlying libraries have been tested on the following platforms:

  • Windows 2000/XP/Vista
  • Solaris
  • Linux
  • Mac OS X
Although untested, this application should work well on Windows Server 2003 and on other platforms supported by Java. For a detailed list of supported system configurations see: http://java.sun.com/j2se/1.5.0/system-configurations.html
